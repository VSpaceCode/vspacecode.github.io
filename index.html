VSpaceCode

Spacemacs like keybindings for Visual Studio Code

Get Started

Mnemonic Bindings

VSpaceCode provides key bindings that are organized by mnemonic key like f for file and p for project and similar to spacemacs.

Batteries Included

VSpaceCode aims to provide the best out-of-the-box experience and bundles all the extensions needed out-of the box.

Customizable

The key binding and its menu are fully customizable. VSpaceCode aims to provide an escape hatch whenever possible to allow customization.
